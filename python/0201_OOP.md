## 1. 인스턴스 클래스 메서드

``` python
class Puppy:
    population = 0
    
    def __init__(self, name, breed = '멍멍이'):
        self.name = name
        self.breed = breed
        Puppy.population += 1
```

```python
p1 = Puppy('초코', '푸들')
p2 = Puppy('댕댕이', '시츄')
```

p1 : 클래스 Puppy의 인스턴스
